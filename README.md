# ResoniteLink experiment

This directory is a local scratchpad for experimenting with **ResoniteLink** (WebSocket + JSON) using raw JSON messages.

## How to navigate

- Start at `notes/INDEX.md`.
- ResoniteLink docs live under `notes/resonitelink/`.
- Resonite docs live under `notes/resonite/`.

## Layout

- `notes/INDEX.md`: knowledge index (ResoniteLink + Resonite categories)
- `notes/resonitelink/INDEX.md`: ResoniteLink-specific docs (connectivity, protocol, recipes)
- `notes/resonite/INDEX.md`: Resonite-specific docs
- `samples/*.json`: copy/paste JSON examples
- `samples/button_interactions/*.json`: button interaction sample set
